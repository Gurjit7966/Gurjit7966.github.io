# Gurjit Singh | Business Analyst Portfolio 🏎️

A high-performance, Formula 1-inspired personal portfolio designed to showcase my work in healthcare analytics, reporting automation, and business intelligence. 

Built with a premium dark aesthetic, this single-page portfolio features cinematic scroll animations, a custom 3D particle system, and dynamic data-driven components.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **3D Graphics:** Three.js (Custom particle system)
* **Animations:** GSAP + ScrollTrigger (Cinematic scroll reveals)
* **Assets:** Optimized `.avif` imagery and custom inline SVGs

## ✨ Key Features
* **Cinematic F1 Aesthetic:** Pure black canvas, racing red accents, and heavy typography inspired by the official Formula 1 design language.
* **3D Hero Section:** A customized Three.js particle field floating over an F1 concept car with a glowing gradient vignette.
* **Team Line Up (Experience):** Work history styled as an F1 constructor lineup, complete with custom SVG cars and team colorways.
* **Live Career Telemetry:** A JavaScript-powered live clock tracking real-time career progression down to the day.
* **Zero Frameworks:** Built entirely with raw vanilla HTML, CSS, and JS (excluding animation libraries) for maximum load performance.
